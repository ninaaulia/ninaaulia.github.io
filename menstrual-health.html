Section 01

Menstrual Health Monitoring & Risk Detection

This project demonstrates how Data Science and Machine Learning can support menstrual health monitoring through predictive analytics and anomaly detection.

Using synthetic menstrual cycle data from 100 users, the system combines four analytical approaches:

  • Exploratory Data Analysis (EDA)
  • Classification Modeling
  • Time-Series Forecasting
  • Anomaly Detection

Key contributions include a risk detection framework, predictive forecasting, and an interactive Streamlit dashboard.

Section 02

Dataset Source & Exploratory Analysis

The dataset contains synthetic menstrual cycle data from 100 users, designed to support predictive modeling and factor analysis in menstrual health research.

EDA examines relationships between:

  • Cycle Length & Period Length
  • Stress Level
  • BMI
  • Sleep Hours
Goal: uncover patterns and correlations that signal cycle irregularities — before any model is trained.

Section 03

Modeling Approaches

Three complementary ML techniques address different dimensions of menstrual health monitoring:

  • Classification — binary target irregular_cycle (Logistic Regression, Random Forest, XGBoost)
  • Forecasting — ARIMA & LSTM predict future cycle length using historical patterns; evaluated with MAE & RMSE
  • Anomaly Detection — Isolation Forest (contamination 0.1) with PCA visualization detects unusual patterns without labeled data
